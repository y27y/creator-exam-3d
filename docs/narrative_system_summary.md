# AI叙事模块实现总结

## 概述

成功构建了一个**涌现式叙事系统**，让AI与玩家共同创造故事。故事不是预先编写的，而是从互动中自然生长。

## 实现架构

### Phase 1: AI叙事引擎后端 (server.js)

新增 `/api/narrative` 端点，支持四种叙事类型：

- **dialogue** - 角色对话生成
- **environment** - 环境描述生成
- **event** - 事件叙事生成
- **lore** - 造物起源故事生成

**关键特性：**
- 复用现有 DeepSeek API 配置
- 动态构建提示词，包含世界状态和上下文
- 叙事风格：诗意、神秘、东方奇幻
- 每次输出100-200字

### Phase 2: 世界状态与记忆系统 (debugGame.js)

**WorldState 追踪：**
- 基础状态：当前关卡、救援/失去人数、裂隙值
- 行为追踪：玩家动作历史、选择记录
- 叙事状态：已发现的故事片段、角色关系、世界事件
- 故事种子：牺牲、拯救、创造、独特策略

**动态分析：**
- `analyzePlayStyle()` - 自动识别玩家风格
  - 自然派（不造东西也能赢）
  - 造物狂（大量造物）
  - 完美主义者（零损失）
  - 牺牲者（付出代价）
  - 平衡型

### Phase 3: AI角色人格系统 (debugGame.js)

**NPCManager 管理6个关卡的独特角色：**

| 关卡 | 角色 | 人格 | 说话风格 |
|------|------|------|----------|
| 洪水村庄 | 老渔夫 | 悲观但善良 | 简短、比喻、常提水和鱼 |
| 洪水村庄 | 小烛 | 天真好奇 | 充满想象力、问很多问题 |
| 永夜矿井 | 矿灯幽灵 | 孤独渴望被记住 | 断断续续，像旧录音 |
| 巨兽困城 | 驯兽人 | 冷静与自然和谐 | 缓慢、深思熟虑、引用古语 |
| 失语战争 | 边境诗人 | 理想主义浪漫 | 诗意、充满隐喻 |
| 记忆瘟疫 | 守忆人 | 固执忠诚 | 重复、像背诵 |
| 终考 | 世界之灵 | 古老疲惫慈悲 | 宏大、像风声 |

**角色特性：**
- 独立记忆系统（记住与玩家的对话）
- 动态态度变化（根据玩家行为调整）
- 情绪状态（忧伤、兴奋、孤独、平静等）

### Phase 4: 涌现式叙事交互层 (cli.js)

**新增命令：**

```
talk, t              - 与当前位置的NPC对话
talk <npcId>         - 与指定NPC对话
explore, ex          - 探索当前环境
remember, mem        - 查看已发现的故事片段
world, wstate        - 查看世界状态与叙事记忆
```

**交互流程：**
1. 玩家输入 `talk`
2. 显示可对话角色列表
3. 选择角色后进入对话循环
4. 玩家输入对话内容
5. AI根据角色人格、世界状态、记忆生成回应
6. 记录到角色记忆和世界状态

## 测试验证

### 对话测试
```bash
curl -X POST http://localhost:3000/api/narrative \
  -H "Content-Type: application/json" \
  -d '{"type":"dialogue","context":{"characterName":"老渔夫"...}}'
```

**返回：**
> "我是这个村子里的老渔夫，打了一辈子鱼，现在打不了啦。你看这水，都漫到我家门槛了。"

### 环境描述测试
```bash
curl -X POST http://localhost:3000/api/narrative \
  -H "Content-Type: application/json" \
  -d '{"type":"environment","context":{"location":"3,3"...}}'
```

**返回：**
> "雨后的平地泛着灰白色的光，积水像一面面破碎的镜子...这片平地即将变成孤岛，而所有声音都被水声压得又低又碎。"

### 造物起源故事测试
```bash

  -H "Content-Type: application/json" \
  -d '{"type":"lore","context":{"creationName":"月光水母"...}}'
```

**返回：**
> "在裂隙之地的第一道伤口处，月光碎成了千片。每一片都化作一只透明的绢丝水母..."

## 涌现式叙事机制

### 1. 记忆累积
- 角色记住与玩家的每次对话
- 世界状态记录玩家的每个选择
- 记忆影响未来的叙事生成

### 2. 态度演化
- 帮助角色 → 态度变友好
- 伤害角色 → 态度变敌对
- 忽略角色 → 态度变失望

### 3. 风格适应
- AI根据玩家行为模式调整叙事风格
- 创意型玩家看到更多诗意描述
- 策略型玩家看到更多实用信息

### 4. 故事种子
- 系统追踪关键事件（牺牲、拯救、创造）
- 这些种子在后续叙事中发芽
- 形成连贯的个人故事线
