# 造物者考核3D (Creator Exam 3D)

> "你是一名裂隙行者——拥有造物能力的人，但你的力量来自世界本身的裂隙。每次造物都会加深裂隙，但也能拯救生命。"

一个AI驱动的3D策略解谜游戏，玩家用**自然语言**创造造物，AI将其编译为游戏能力，在7x7的3D格子地图上解决危机、拯救生命、理解世界的真相。

---

## 核心特性

### 自然语言造物系统
- 输入任意自然语言描述，AI将其编译为结构化游戏卡牌
- 示例："创造一群会把洪水搬到天空的透明鲸鱼" → 生成吸水能力卡牌
- 本地离线编译器兜底，无需AI API即可运行

### AI驱动叙事引擎
- **4种AI Storyteller人格**：卡珊德拉（悲观预警）、菲比（乐观希望）、兰迪（混沌随机）、说书人（平衡叙事）
- **涌现式世界传说**：基于玩家行为动态生成神话故事
- **叙事因果引擎**：蝴蝶效应系统，微小选择产生长期影响
- **语义向量记忆**：50维TF-IDF追踪玩家行为模式
- **认知效果系统**：文本扭曲、矛盾对话、记忆碎片、既视感

### 深度策略系统
- **13种连锁反应组合**：光+水=蒸汽、土+水=沼泽、时间+路径=预知...
- **仪式熔炉**：8种配方，4种原型，涌现式仪式发现
- **言灵誓约**：5种誓约类型，背叛概率计算，怨恨链机制
- **认知深渊**：高熵混沌模式，谜题解码，NPC反转
- **验证腐化**：悖论卡牌，引擎过载，腐化传播
- **造物者工坊**：拆解/改造/融合/注入，8种材料，6种改造
- **敌方意图预览**：Into the Breach风格完美信息，提前显示敌人意图
- **长夜守城**：第6到第7天之间的塔防过场，读取最多18张造物卡和近期经历，用 DeepSeek Flash 等模型生成守夜塔名称、因果简报、三选一守夜加成、数值偏向、射程与技能说明
- **第七天裂隙空域**：终局 Canvas 空战清算层，造物武器化，6段Boss航线回写最终结局

### NPC与社交系统
- **动态情绪系统**：8种情绪状态，情绪传染机制
- **社交图谱**：6种关系类型，小团体检测，社交距离计算
- **单位传承**：Wildermyth风格，8种特质，4个传承等级
- **跨关卡持久化**：被救援单位在未来关卡重现，关系状态继承

### 3D视觉体验
- Three.js低多边形渲染，InstancedMesh优化
- 粒子特效系统（能力激活、救援/失败、环境效果）
- 屏幕特效（闪光、震动、暗角、颜色分级）
- 动态路径可视化

## 快速开始

### 环境要求
- Node.js >= 18
- 现代浏览器（支持ES Modules和WebGL）

### 安装运行

```bash
# 克隆项目
cd creator_exam_3d

# 启动服务器
npm start

# 打开浏览器访问
http://localhost:3000
```

> 项目无npm依赖，`npm start` 直接执行 `node server.js`。Three.js通过CDN加载。

### 接入AI（可选）

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件
AI_API_KEY=你的密钥
AI_BASE_URL=https://你的-openai-compatible-endpoint/v1
AI_MODEL=你的模型名
PORT=3000
```

DeepSeek Flash 等 OpenAI-compatible 模型可通过 `AI_BASE_URL` / `AI_MODEL` 接入同一套代理。空战、NPC对话和主线叙事共用 `/api/narrative`；长夜守城额外使用 `/api/night-watch-towers`，由此前造物、居民、近期经历、熵值和守夜结果提供上下文。未配置密钥时，该端点使用本地兜底生成不同造物对应的守夜塔、因果说明和三选一守夜加成。

未配置AI时，游戏使用本地离线编译器和本地叙事兜底，仍可完整运行。

## 玩法说明

1. **观察**左侧关卡目标和当前危机状态
2. **创造**在右侧输入造物想法，例如：
   - "造一棵会发光并指路的月亮树"
   - "让两族的梦境在同一张桌子上相遇"
   - "创造一面吸收所有悲伤的镜子"
3. **生成**点击"生成造物卡"，AI编译为游戏能力
4. **放置**点击"放置造物"，在3D地图上选择格子
5. **结算**点击"结束回合"，观察灾害扩散、单位移动和连锁反应
6. **目标**达成关卡目标进入下一关，失败则重来

### 能力池
- **水系**：吸水、造桥、冻结
- **光系**：照明、路径显示、太阳祝福
- **地形系**：改造、抬升、种植森林、挖掘水渠
- **防御系**：墙壁、力场、陷阱
- **精神系**：安抚、引导、记忆信标、梦境连接
- **特殊**：净化、时间延缓

## 核心系统详解

### 1. AI Storyteller ( storyteller.js )
参考RimWorld的AI故事叙述者，4种人格动态调整游戏事件：
- **卡珊德拉**：事件频率递增，制造持续压力
- **菲比**：事件间隔较长，给予玩家喘息空间
- **兰迪**：完全随机，可能连续灾难或长期和平
- **说书人**：平衡型，根据玩家表现动态调整

### 2. 连锁反应系统 ( chainReactionCodex.js )
13种能力组合产生额外效果：
| 组合 | 效果 |
|------|------|
| 照明 + 吸水 | 蒸汽迷雾（临时阻挡视野） |
| 森林 + 屏障 | 不可破坏屏障 |
| 引导 + 记忆信标 | 被引导单位免疫混乱 |
| 陷阱 + 迟缓 | 巨兽额外停留 |
| 时间 + 路径 | 预知（显示未来3回合） |
| 净化 + 力场 | 圣域（双重效果） |
| 梦境 + 记忆 | 永恒连接（不随时间断开） |

### 3. 敌方意图预览 ( enemyIntent.js )
Into the Breach风格：每回合开始前显示所有敌人的意图：
- 巨兽：前进/冲锋/暴走/受阻/被困
- 平民：徘徊/前进/受引导/混乱/梦境连接
- 使者：前进/误判/受引导
- 灾害：扩散/加剧/休眠

### 4. 认知深渊 ( cognitiveAbyss.js )
高熵游戏层，当裂隙值超过阈值时触发：
- 谜题解码（凯撒密码、反向文本、符号替换）
- NPC反转（敌人变友军）
- 悖论卡牌（非法能力产生意外效果）
- 深渊深度系统（越深入奖励越高风险越大）

### 5. 造物者工坊 ( creatorWorkshop.js )
关卡之间的资源管理空间：
- **拆解**：将造物还原为材料
- **改造**：消耗材料改变造物属性（范围+1、持续时间+2等）
- **融合**：两个造物合成新造物（特殊配方产生稀有能力）
- **注入**：将传承单位的特质注入造物
