# 连贯叙事框架设计

## 问题分析

当前6个关卡完全割裂：
- 第1关：洪水村庄（村民：阿粟、小烛、木匠、邮差）
- 第2关：永夜矿井（矿工甲、乙、丙、丁）
- 第3关：巨兽困城（古水巨兽）
- 第4关：失语战争（东岸使者、西岸使者）
- 第5关：记忆瘟疫（南枝、北声、阿眠、谷雨、织火）
- 第6关：终考（星砂、青麦、砾歌、裂隙兽、北境使者、南境使者）

**问题：**
1. 每关人物完全不同，玩家无法建立情感连接
2. 没有统一的世界观解释为什么这些灾难会发生
3. 玩家的选择不会影响到后续关卡
4. 缺乏一个贯穿始终的主角或目标

## 解决方案：统一叙事框架

### 核心设定：造物者的试炼

**玩家身份：** 你是一名"裂隙行者"——拥有造物能力的人，但你的力量来自世界本身的裂隙。每次造物都会加深裂隙，但也能拯救生命。

**世界设定：** 
- 世界曾经完整，但"大分离"事件让现实出现了裂缝
- 裂缝中涌出了各种灾难（洪水、黑暗、战争、遗忘）
- 你的使命不是消灭灾难，而是理解它们、与它们共存
- 最终目标是找到"缝合裂隙"的方法，而不是简单地阻止灾难

### 连贯故事线：七天创世

把6关重新包装为"七天创世"的试炼：

**序章：觉醒**
- 你醒来在一个陌生的世界，听到一个声音（世界之灵）告诉你：你是造物者，必须在七天内学会创造与毁灭的平衡

**第一天：水（洪水村庄）**
- 地点：临水村
- 角色：阿粟、小烛、木匠、邮差
- 主题：学会"引导"而非"控制"
- 关键NPC：老渔夫（他知道洪水是世界的眼泪）
- 结局：你救了村民，但老渔夫告诉你："水不是敌人，它在寻找回家的路"

**第二天：暗（永夜矿井）**
- 地点：临水村下方的旧矿井
- 角色：阿粟（自愿下来找你）、矿工们
- 主题：学会"照亮"而非"驱逐"
- 关键NPC：矿灯幽灵（他是上一任造物者）
- 连接：洪水渗入地下，导致矿井黑暗扩散
- 结局：幽灵告诉你："黑暗不是虚无，是未被倾听的声音"

**第三天：兽（巨兽困城）**
- 地点：临水村通往城市的道路
- 角色：古水巨兽（其实是老渔夫的旧识）、小烛（偷偷跟来）
- 主题：学会"安抚"而非"征服"
- 关键NPC：驯兽人
- 连接：巨兽是被分离事件扭曲的自然之灵，它在寻找水源
- 结局：你理解了巨兽的悲伤，它背上的水源是治愈裂隙的关键

**第四天：言（失语战争）**
- 地点：临水村与邻村的边界
- 角色：东岸使者、西岸使者（其实是阿粟和邮差的族人）
- 主题：学会"倾听"而非"说服"
- 关键NPC：边境诗人
- 连接：前两天的灾难让两个村子互相指责，语言变成了武器
- 结局：你发现"星火"其实是世界裂隙的光芒，两个村子都在保护它

**第五天：忆（记忆瘟疫）**
- 地点：圣树森林（世界裂隙的源头）
- 角色：南枝、北声、阿眠、谷雨、织火（都是前几关角色的亲人）
- 主题：学会"记住"而非"遗忘"
- 关键NPC：守忆人（他是世界之灵的碎片）
- 连接：裂隙在扩大，开始吞噬记忆，前几关的经历变得模糊
- 结局：你意识到记忆是连接世界的纽带，不能为了逃避痛苦而遗忘

**第六天：终考（第七天之前）**
- 地点：裂隙核心
- 角色：所有前5关的角色汇聚
- 主题：学会"平衡"——创造与毁灭、拯救与牺牲
- 关键NPC：世界之灵（揭示真相）
- 真相：你本身就是裂隙的一部分，你的造物能力来自世界的伤口。只有接受这一点，才能真正治愈世界
- 结局：多种结局取决于前5关的选择

### 角色连续性设计

**核心角色贯穿全程：**

1. **阿粟**（第1关村民）
   - 第1关：普通村民，被洪水威胁
   - 第2关：勇敢下来矿井找你，成为向导
   - 第3关：理解巨兽的语言，成为翻译
   - 第4关：作为东岸使者，试图阻止战争
   - 第5关：开始遗忘，但记得你的脸
   - 第6关：汇聚所有记忆，帮助你理解真相

2. **小烛**（第1关小孩）
   - 第1关：天真好奇，能看到造物之光
   - 第2关：偷偷跟来，在黑暗中不害怕
   - 第3关：与巨兽成为朋友，理解它的悲伤
   - 第4关：用孩子的纯真化解大人的仇恨
   - 第5关：第一个记起事情的人，因为孩子的记忆最纯净
   - 第6关：代表"希望"，她的存在证明世界值得拯救

3. **世界之灵**（贯穿全程的神秘声音）
   - 第1关：只在序章出现，告诉你基本规则
   - 第2关：通过矿灯幽灵传达信息
   - 第3关：通过巨兽的梦境出现
   - 第4关：通过诗人的诗歌出现
   - 第5关：通过守忆人出现
   - 第6关：直接现身，揭示全部真相

### 选择影响系统

**前5关的选择会影响第6关：**

- 第1关救了多少人 → 第6关有多少盟友
- 第2关是否理解了黑暗 → 第6关能否利用黑暗的力量
- 第3关是否安抚了巨兽 → 第6关巨兽是敌是友
- 第4关是否阻止了战争 → 第6关两个村子是否合作
- 第5关是否保留了记忆 → 第6关你是否记得自己的使命

**多种结局：**

1. **完美结局**：平衡之道（救了所有人，理解了所有灾难，缝合裂隙）
2. **牺牲结局**：你成为新的世界之灵，替代旧的
3. **遗忘结局**：你选择遗忘一切，世界继续崩解，但你不痛苦
4. **创造结局**：你创造了新的世界，但旧世界彻底消失
5. **毁灭结局**：裂隙吞噬一切，但这是重生的开始

### 实现方案

**需要修改的文件：**

1. `levels.js` - 重写关卡故事，添加角色关联
2. `debugGame.js` - 添加跨关卡状态传递
3. `server.js` - 叙事引擎支持跨关卡上下文
4. `cli.js` - 添加序章和结局流程

**技术实现：**

```javascript
// 跨关卡状态传递
class GameSession {
  constructor() {
    this.levelResults = []; // 每关的结果
    this.characterRelationships = {}; // 角色关系
    this.discoveredTruths = []; // 发现的真相
    this.playerChoices = []; // 玩家选择
  }
  
  // 进入下一关时传递状态
  getContextForLevel(levelId) {
    return {
      previousResults: this.levelResults,
      characterStates: this.characterRelationships,
      truths: this.discoveredTruths,
      choices: this.playerChoices
    };
  }
}
```
