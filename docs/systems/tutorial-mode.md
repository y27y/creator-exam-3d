# 完整教学战役

## 目标

教学模式覆盖六关地面主线、长夜守城、第七日裂隙空域和最终结算。玩家跟随当前提示即可走完主线；高级系统不是一次性打卡，而是在真实棋盘上重复使用。

## 玩家交互

- 首次进入可选择“开始完整教学”或自由考核。
- 主界面右侧轨道为 `人 / 志 / 术 / 教`。“教”展开当前一步，再次点击或切换其他抽屉时折叠。
- 点击“教”中的高级系统动作会直接在当前棋盘结算，不再反复自动展开“术”；玩家手动打开“术”时，顶部会优先显示同一教学动作，并列出本课已经放置的固定造物。
- 教学卡只保留当前动作、必要的推荐描述和动作按钮，不显示列／行坐标或大段放置理由。
- 生成与放置阶段，目标格直接显示金色圆环和箭头；放错位置会拒绝落子并提示点击金色标记。
- 推荐描述保持诗性，但包含明确的对象、作用与目标。使用推荐描述时直接载入对应的固定教学造物，不请求云端；玩家自行改写的描述仍走云端／本地编译，并在能力兼容时接受教学校准。
- “重新开始本关”只清空当前关教学步骤、检查点与棋盘状态，保留其他章节进度；“重置全部教学”经过二次确认后清空整条教学战役进度并返回第一关，不删除普通存档。

| 章节 | 推荐造物 | 主要能力 | 场景指示 |
| --- | --- | --- | --- |
| 洪水村庄 | 饮潮莲 | 吸水并显路 | 棋盘金色标记 |
| 永夜矿井 | 轨道月树 | 照明并显路 | 棋盘金色标记 |
| 巨兽困城 | 潮声骨笛 | 安抚巨兽 | 棋盘金色标记 |
| 失语战争 | 停战路标 | 显路、引导、平息误判 | 棋盘金色标记 |
| 记忆瘟疫 | 唤名信标 | 唤回目标与方向 | 棋盘金色标记 |
| 第七天之前 | 三向风门、终考路光 | 改道灾害、牵制裂隙兽、显路并稳定会谈 | 两个连续金色标记 |

这些复合承诺通过卡牌上的 `tutorialEffects` 明示，不修改普通卡牌，也不绕过正常回合结算。记忆瘟疫与终考的黄金卡额外使用 `tutorialEffectsGlobal`，在教学回合里为全部在途居民／使者持续显路和引导，避免远处单位先进入迷雾后被随机方向带离黄金路线。黄金卡生效期间仍会出现“顿悟、巨兽犹豫、奇迹共鸣”等正向随机事件，但地震、额外暴雨、裂隙增幅、随机改道和旁白挫折只保留叙事反馈，不改写棋盘与裂隙值，防止教学路线被流程外的随机因素破坏。

## 高级系统覆盖

教学共安排 31 次高级系统步骤。敌方意图、连锁反应、言灵誓约和工坊会跨多个章节重复使用；仪式、深渊谜题、裂隙回响、验证腐化、说书人、世界传说、社交图谱、传承、居民对话和独立存档也会至少实际体验一次。

系统动作由 `TutorialDirector.executeSystem()` 调用现有入口，并且只在真实动作返回成功后记录完成。每次关键放置、高级动作和回合结算前都会写入检查点；失败时优先恢复最近检查点，也可以重置本课。

## 最终模式

### 长夜守城

- 固定使用已验证地图，教学资源和塔数值降低失败压力。
- 三个塔位直接显示为地图 1/2/3 金色标记，放塔后对应标记熄灭。
- 教学卡可折叠为“教”。正常路线要求按标记放塔并优先升级 1 号塔。
- 若玩家仍无法通过，可点击“教学保底 · 全塔裁决”清除当前敌人。

### 第七日裂隙空域

- 教学航线降低敌机与弹幕压力、缩短 Boss 血量，并在致命伤时保留载体。
- 教学提示位于右上，可折叠为“教”；技能和保底按钮保留在右下。
- 正常路线要求保持在画面下方、横移躲开紫色预警并重复使用造物脉冲。
- 若玩家仍无法通过，可点击“教学保底 · 开启无敌”；它会回满生命、清除敌弹并永久免疫本局伤害。

两个保底按钮都是给玩家的教学外挂，但不计入课程步骤、高级系统覆盖或无外挂通关验收。

## 验收

```bash
npm run test:tutorial
npm run check
node debug/browser-modes-smoke.js
node debug/current-code-reality-tests.js
node debug/test-suite.js
```

`tutorial-campaign-tests` 使用 64 个确定随机种子逐回合运行六关黄金路线，共验证 384 局且必须全部胜利；同时确认七张推荐造物可被确定性重建。视觉验收还需要在窄屏检查：四个右侧按钮不被教学卡覆盖、教学可折叠、目标标记可见、守夜标记随放置消失，以及空战教学／技能／保底都完整落在一屏内。
